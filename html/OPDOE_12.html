design.regression R Documentation

Regression Design Object

Description

An design.regression object is created with design.regression.polynom

Arguments

A triangular.test object is a list of

model

character, currently only "polynomial" is implemented

locations

choosen locations
replications

choosen replications per location
interval

vector of size 2 storing the given interval

Author(s)

Dieter Rasch, Juergen Pilz, L.R. Verdooren, Albrecht Gebhardt

References
