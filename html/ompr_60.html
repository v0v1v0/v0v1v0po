variable_keys R Documentation

Get all unique names of the model variables

Description

Get all unique names of the model variables

Usage

variable_keys(model)

Arguments

model

the model

Value

a character vector ordered in the same way as the constraint matrix columns and objective vector

Examples

library(magrittr)
model <- MIPModel() %>%
  add_variable(x[i], i = 1:3)
variable_keys(model)
