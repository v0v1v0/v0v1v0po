Write text to a file

Usage

  header = "**THIS FILE IS CONSTANTLY OVERWRITTEN -- DO NOT MANUALLY EDIT**\r\n\r\n"
)

Arguments

txt

Text to be written
file

header

Optional header that is inserted at the top of the text file
