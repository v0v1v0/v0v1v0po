notIn R Documentation

Opposite of '%in%'

Description

Indicate if elements of a vector are not in another vector.

Usage

notIn(x, table)

x %notin% table

Arguments

x, table

Vectors.

Value

A logical vector.

Examples


x <- c('a', 'v', 'o', 'C', 'a', 'd', 'O')
y <- letters

y %notin% x
x %notin% y
