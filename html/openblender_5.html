genericDownloadCall R Documentation

Request to the API, depending on the action provided

Description

Prepare the data to send it 'OpenBlender' API. This function is not used by users.

Usage

genericDownloadCall(
  json_parametros,
  url,
  action,
  n_test_observations,
  slice_mult
)

Arguments

json_parametros

Request parameters
url

Url selected

Value

List of observations obtained with dameRespuestaLlamado.
