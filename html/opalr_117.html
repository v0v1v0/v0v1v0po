opal.command R Documentation

Get an asynchronous command

Description

Get an asynchronous R commands in the remote R session.

Usage

opal.command(opal, id, wait = FALSE)

Arguments

opal

Opal object.
id

R command ID.
wait

Wait for the command to complete.

See Also

Other command functions: opal.command_result(), opal.command_rm(), opal.commands(), opal.commands_rm()

Examples

## Not run: 
o <- opal.login('administrator','password', url='https://opal-demo.obiba.org')
opal.command(o, '1234')
opal.logout(o)

## End(Not run)
