theme_default R Documentation

A ggplot2 theme

Description

A simple theme for ggplot2 that loosely resembles nicely themed plots from base graphics.

Usage

theme_default(
  base_size = 11,
  base_family = "",
  text_col = "grey20",
  panel_border_col = "grey70"
)

Arguments

base_size

Base font size.
base_family

Base font family.
text_col

Color for text.
panel_border_col

Color for panel borders.

Examples

p <- ggplot2::ggplot(mtcars) +
  ggplot2::geom_point(ggplot2::aes(x = wt, y = mpg, colour = factor(gear))) +
  ggplot2::facet_wrap(~am)
p + theme_default()
