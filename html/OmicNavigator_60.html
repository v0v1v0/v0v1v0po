group R Documentation

group from Bioconductor workflow RNAseq123

Description

A subset of the object group from Bioconductor workflow RNAseq123.

Usage

group

Format

A factor with 3 levels:

Basal

Basal cells

LP

Luminal progenitor cells

ML

Mature luminal cells
