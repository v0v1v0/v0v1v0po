residuals_o R Documentation

The orthogonal residuals

Description

Returns a vector with the orthogonal residuals \| D_i \| from the fitted onls model.

Usage

residuals_o(object)

Arguments

object

an object returned from onls.

Value

The orthogonal residuals.
