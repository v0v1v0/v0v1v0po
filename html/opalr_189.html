opal.table_delete R Documentation

Delete a Opal table

Description

Removes both values and data dictionary of a table, or remove the table's logic if the table is a view. Fails if the table does not exist. See also opal.table_truncate.

Usage

opal.table_delete(opal, project, table, silent = TRUE)

Arguments

opal

Opal connection object.
project

Project name where the table is located.
table

Table name to be deleted.
silent

Warn if table does not exist, default is TRUE.

See Also

Other table functions: opal.perms(), opal.resource_view_create(), opal.resource_view_reconnect(), opal.table_create(), opal.table_dictionary_get(), opal.table_dictionary_update(), opal.table_exists(), opal.table_export(), opal.table_get(), opal.table_import(), opal.table_perm(), opal.table_perm_add(), opal.table_perm_delete(), opal.table_save(), opal.table_truncate(), opal.table_view_create(), opal.table_view_update()

Examples

## Not run: 
o <- opal.login('administrator','password', url='https://opal-demo.obiba.org')
opal.table_delete(o, "CNSIM", "CNSIM1")
opal.logout(o)
