Usage

blockboot(
  ts,
  R,
  seed,
  n_cores,
  methods = c("optnbb", "optmbb", "optcbb", "opttmbb", "opttcbb")
)

ts

number of resample

RNG seed
n_cores

number of core(s) to be used on your operaterating system

Functions

Examples

  set.seed(289805)
  ts <- arima.sim(n = 3, model = list(ar = 0.8, order = c(1, 0, 0)), sd = 1)
  blockboot(ts = ts, R = 2, seed = 6, n_cores = 1)

  set.seed(289805)
  ts <- arima.sim(n = 3, model = list(ar = 0.8, order = c(1, 0, 0)), sd = 1)
