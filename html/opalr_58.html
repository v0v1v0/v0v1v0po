dsadmin.profile_perm R Documentation

Get the permissions of a DataSHIELD profile

Description

Get the permissions of a DataSHIELD profile

Usage

dsadmin.profile_perm(opal, name)

Arguments

opal

Opal connection object.
name

Profile name.

Value

A data.frame with columns: subject, type, permission

See Also

Other DataSHIELD profiles: dsadmin.profile(), dsadmin.profile_access(), dsadmin.profile_create(), dsadmin.profile_delete(), dsadmin.profile_enable(), dsadmin.profile_exists(), dsadmin.profile_init(), dsadmin.profile_perm_add(), dsadmin.profile_perm_delete(), dsadmin.profile_rparser(), dsadmin.profiles()

Examples

## Not run: 
o <- opal.login('administrator','password', url='https://opal-demo.obiba.org')
dsadmin.profile_perm_add(o, 'default', c('andrei', 'valentina'), 'user', 'use')
dsadmin.profile_perm(o, 'default')
dsadmin.profile_perm_delete(o, 'default', c('andrei', 'valentina'), 'user')
opal.logout(o)

## End(Not run)
