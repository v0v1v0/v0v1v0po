clone_oolong R Documentation

Clone an oolong object

Description

Clone a new oolong object. The oolong must not be locked and ever coded.

Usage

clone_oolong(oolong, userid = NA)

Arguments

oolong

an oolong object.
userid

a character string to denote the name of the coder

Value

an oolong object

Author(s)

Chung-hong Chan
