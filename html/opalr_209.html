opal.taxonomy_delete R Documentation

Delete a taxonomy

Description

Delete a taxonomy, without failing if the taxonomy does not exist.

Usage

opal.taxonomy_delete(opal, taxonomy)

Arguments

opal

Opal object.
taxonomy

Name of the taxonomy.

See Also

Other taxonomy functions: opal.taxonomies(), opal.taxonomy(), opal.taxonomy_download(), opal.taxonomy_upload(), opal.terms(), opal.vocabularies(), opal.vocabulary()

Examples

## Not run: 
o <- opal.login('administrator','password', url='https://opal-demo.obiba.org')
opal.taxonomy_delete(o, 'Mlstr_area')
opal.logout(o)

## End(Not run)
