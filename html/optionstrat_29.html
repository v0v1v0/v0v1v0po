tdiff R Documentation

Time Difference

Description

Computes the difference in time between two dates

Usage

tdiff(date1, date2, period = c("days, years"))

Arguments

date1

Earlier date
date2

Later date
period

String value, either "days", or "years"

Value

Returns a numeric value

Examples

tdiff("2018-01-01", "2018-06-30", "days")
