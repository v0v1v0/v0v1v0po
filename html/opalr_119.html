opal.commands_rm R Documentation

Remove all asynchronous commands

Description

Remove all asynchronous R commands in the remote R session.

Usage

opal.commands_rm(opal)

Arguments

opal

Opal object.

See Also

Other command functions: opal.command(), opal.command_result(), opal.command_rm(), opal.commands()

Examples

## Not run: 
o <- opal.login('administrator','password', url='https://opal-demo.obiba.org')
opal.commands_rm(o)
opal.logout(o)

## End(Not run)
