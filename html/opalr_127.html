opal.file R Documentation

Get file content

Description

Get file content from the Opal file system.

Usage

opal.file(opal, path, key = NULL)

Arguments

opal

Opal object.
path

Path to the file in the Opal file system.
key

File encryption key: downloaded file will be a zip file with content encrypted (use 7zip to decrypt).

See Also

Other file functions: opal.file_cp(), opal.file_download(), opal.file_ls(), opal.file_mkdir(), opal.file_mkdir_tmp(), opal.file_mv(), opal.file_read(), opal.file_rm(), opal.file_unzip(), opal.file_upload(), opal.file_write()

Examples

## Not run: 
o <- opal.login('administrator','password', url='https://opal-demo.obiba.org')
opal.file(o, '/home/administrator/joins/join-src-3.csv')
opal.logout(o)

## End(Not run)
