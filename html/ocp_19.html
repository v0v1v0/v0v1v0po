Print out ocpd object summary.

Usage

## S3 method for class 'ocp'
summary(object, ...)

Arguments

object

the object to summarize
...

(optional) additional arguments, ignored.

Examples

simdatapts<- c(rnorm(n = 50), rnorm(n=50, 100))
