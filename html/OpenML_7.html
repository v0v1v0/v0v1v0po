convertOMLFlowToMlr R Documentation

Converts a flow to a mlr learner.

Description

Converts an OMLFlow that was originally created with the OpenML R-package to a Learner.

Usage

convertOMLFlowToMlr(flow)

Arguments

flow

[OMLFlow]
The flow object.

Value

[Learner].
