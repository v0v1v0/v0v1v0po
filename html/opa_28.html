random_pccs R Documentation

Return the random order generated PCCs used to calculate the group chance value

Description

Return the random order generated PCCs used to calculate the group chance value

Usage

random_pccs(m)

Arguments

m

an object of class "opafit"

Value

a numeric vector

Examples

dat <- data.frame(t1 = c(9, 4, 8, 10),
                  t2 = c(8, 8, 12, 10),
                  t3 = c(8, 5, 10, 11))
h <- hypothesis(1:3)
opamod <- opa(dat, h)
random_pccs(opamod)
