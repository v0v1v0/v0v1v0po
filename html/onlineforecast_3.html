==.data.list R Documentation

Determine if two data.lists are identical

Description

Compare two data.lists

Usage

## S3 method for class 'data.list'
x == y

Arguments

x

first data.list
y

second data.list

Details

Returns TRUE if the two data.lists are fully identical, so all data, order of variables etc. must be fully identical

Value

logical

Examples


Dbuilding == Dbuilding

D <- Dbuilding
D$Ta$k2[1] <- NA
Dbuilding == D

D <- Dbuilding
names(D)[5] <- "I"
names(D)[6] <- "Ta"
Dbuilding == D
