opal.task_wait R Documentation

Wait for a task to complete.

Description

The task completion is defined by its status: *SUCCEEDED*, *FAILED* or *CANCELED*.

Usage

opal.task_wait(opal, id, max = NULL)

Arguments

opal

Opal object.
id

Task identifier.
max

Maximum time (in seconds) to wait for the task completion. Default is NULL (no maximum).

See Also

Other task functions: opal.task(), opal.task_cancel(), opal.tasks()

Examples

## Not run: 
o <- opal.login('administrator','password', url='https://opal-demo.obiba.org')
opal.task_wait(o, '1')
opal.logout(o)

## End(Not run)
