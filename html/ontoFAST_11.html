get_onto_name R Documentation

Get names for ontology IDs

Description

Returns names of ontology terms for ontology IDs

Usage

get_onto_name(vec, onto, names = FALSE)

Arguments

vec

ID or a vector of IDs
onto

ontology
names

use element name

Value

vector of names.

Examples

data(HAO)
get_onto_name("HAO:0002272", HAO)
