opal.tables R Documentation

Get tables of a datasource

Description

Get tables of a datasource

Usage

opal.tables(opal, datasource, counts = FALSE, df = TRUE)

Arguments

opal

Opal object.
datasource

Name of the datasource.
counts

Flag to get the number of variables and entities (default is FALSE).
df

Return a data.frame (default is TRUE)

See Also

Other datasource functions: opal.annotate(), opal.annotations(), opal.attribute_values(), opal.datasource(), opal.datasources(), opal.table(), opal.valueset(), opal.variable(), opal.variable_summary(), opal.variables()

Examples

## Not run: 
o <- opal.login('administrator','password', url='https://opal-demo.obiba.org')
opal.tables(o, 'CNSIM')
opal.logout(o)
