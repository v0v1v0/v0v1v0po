pcc_plot R Documentation

Plot individual PCCs.

Description

Plot individual PCCs.

Usage

pcc_plot(m)

Arguments

m

an object of class "opafit"

Value

No return value, called for side effects.

Examples

dat <- data.frame(t1 = c(9, 4, 8, 10),
                  t2 = c(8, 8, 12, 10),
                  t3 = c(8, 5, 10, 11))
h <- hypothesis(1:3)
opamod <- opa(dat, h)
pcc_plot(opamod)
