Get a resource reference of a project

Description

Get a resource reference of a project

Usage

opal.resource(opal, project, resource)

Arguments

opal

Opal object.
project

Name of the project.
resource

Name of the resource in the project.

opal.resource(o, 'RSRC', 'CNSIM1')
