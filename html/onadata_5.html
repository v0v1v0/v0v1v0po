chinook_employees R Documentation

Chinook employee data

Description

Extract of data on employees of a music sales company

Usage

chinook_employees

Format

A dataframe with 8 rows and 4 variables:

EmployeeId

Employee ID number

FirstName

Employee First Name

LastName

Employee Last Name

ReportsTo

ID of Employee who they report to

Source

Chinook Database on Github

Examples

chinook_employees
