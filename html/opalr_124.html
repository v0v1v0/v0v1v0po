opal.delete R Documentation

Generic REST resource deletion.

Description

Generic REST resource deletion.

Usage

opal.delete(opal, ..., query = list(), callback = NULL)

Arguments

opal

Opal object.
...

Resource path segments.
query

Named list of query parameters.
callback

A callback function to handle the response object.

See Also

Other REST functions: opal.get(), opal.post(), opal.put()

Examples

## Not run: 
o <- opal.login('administrator','password', url = 'https://opal-demo.obiba.org')
opal.delete(o, 'some', 'resource')
opal.logout(o)

## End(Not run)
