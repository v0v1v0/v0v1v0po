opal.command_rm R Documentation

Remove an asynchronous command

Description

Remove an asynchronous R commands in the remote R session.

Usage

opal.command_rm(opal, id)

Arguments

opal

Opal object.
id

R command ID.

See Also

Other command functions: opal.command(), opal.command_result(), opal.commands(), opal.commands_rm()

Examples

## Not run: 
o <- opal.login('administrator','password', url='https://opal-demo.obiba.org')
opal.command_rm(o, '1234')
opal.logout(o)

## End(Not run)
