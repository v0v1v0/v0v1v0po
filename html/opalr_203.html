opal.task R Documentation

Get a task

Description

Get the details of a specific task.

Usage

opal.task(opal, id)

Arguments

opal

Opal object.
id

Task identifier.

See Also

Other task functions: opal.task_cancel(), opal.task_wait(), opal.tasks()

Examples

## Not run: 
o <- opal.login('administrator','password', url='https://opal-demo.obiba.org')
opal.task(o, '1')
opal.logout(o)

## End(Not run)
