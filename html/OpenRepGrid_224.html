trim_val R Documentation

trim vector to lower or upper value

Description

the range a value may take is restricted to by an upper and lower boundary. If it exceeds the boundary the value is replaced by the boundary value or alternatively by NA

Usage

trim_val(x, minmax = c(-Inf, Inf), na = FALSE)

Arguments

x

numeric vector
minmax

minimal and maximal possible value (default c(-Inf, Inf) i.e. no trimming occurs)
na

Use NAs for replacing values that are out of range

Value

vector vector whose elements that are out of range are replaced

Examples

trim_val(30)
trim_val(30, c(10, 20))
