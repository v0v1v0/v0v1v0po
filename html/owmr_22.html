owmr_settings R Documentation

owmr settings.

Description

Set api key. Internally it calls Sys.setenv to store the api key in an environment variable called OWM_API_KEY.

Usage

owmr_settings(api_key)

Arguments

api_key

owm api key

Examples

## Not run: 
   owmr_settings(api_key = "your-api-key")

## End(Not run)
