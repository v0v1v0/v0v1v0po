OVL.LogitD

Description

Parametric approach using the delta method after switching to a logit scale and then transforming back

Usage

OVL.LogitD(x, y, alpha = 0.05)

Arguments

x

controls
y

cases
alpha

confidence level

Value

confidence interval

Examples

controls = rnorm(50,6,1)
cases = rnorm(100,6.5,0.5)
OVL.LogitD (controls,cases)
