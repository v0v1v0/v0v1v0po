opal.projects_databases R Documentation

Get projects databases

Description

When creating a project for storing data, it is required to name the database to be associated.

Usage

opal.projects_databases(opal)

Arguments

opal

Opal object.

Value

A character vector of databases names.

See Also

Other project functions: opal.project(), opal.project_create(), opal.project_delete(), opal.project_exists(), opal.projects()

Examples

## Not run: 
o <- opal.login('administrator','password', url='https://opal-demo.obiba.org')
opal.projects_databases(o)
opal.logout(o)

## End(Not run)
