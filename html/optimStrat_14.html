varp R Documentation

Variance

Description

Compute the variance of x.

Usage

varp(x)

Arguments

x

a numeric vector.

Details

Compute the variance of x using n (instead of n-1 as in var) in the denominator.

Value

An object with the variance of x.

See Also

var

Examples

x<- rnorm(100)
varp(x)
var(x)
