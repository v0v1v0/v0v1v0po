opal.unload_package R Documentation

Unload package

Description

Unload package from the remote R session.

Usage

opal.unload_package(opal, pkg)

Arguments

opal

Opal object or list of opal objects.
pkg

Package name.

See Also

Other execution functions: opal.execute(), opal.execute.source(), opal.load_package()

Examples

## Not run: 
o <- opal.login('administrator','password', url='https://opal-demo.obiba.org')
opal.unload_package(o, 'stats')
opal.logout(o)

## End(Not run)
