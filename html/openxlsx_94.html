ungroupColumns R Documentation

Ungroup Columns

Description

Ungroup a selection of columns

Usage

ungroupColumns(wb, sheet, cols)

Arguments

wb

A workbook object
sheet

A name or index of a worksheet
cols

Indices of columns to ungroup

Details

If column was previously hidden, it will now be shown

Joshua Sturm
