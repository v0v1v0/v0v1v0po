labelcora R Documentation

True clustering labels for Cora data set.

Description

True clustering labels for the Cora dataset, which can be applied to 4 views.

Usage

data("labelcora")

Format

The format is: chr [1:2708] "1" "2" "3" "3" "4" "4" "5" "1" "1" "5" "1" "6" "4" "7" ...

Details

True clustering labels for the Cora dataset, which can be applied to 4 views.

References
