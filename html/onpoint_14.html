split_at R Documentation

split_at

Description

Split vector

Usage

split_at(x, pos)

Arguments

x

vector with positions to split.

Details

Split vector at position(s). Returns a list with all elements before and after the split position.

Value

list

Examples

## Not run: 
set.seed(42)
x <- sample(x = 1:10, size = 5)
split_at(x = x, pos = 3)

## End(Not run)
