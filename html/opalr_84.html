oadmin.log R Documentation

Get Opal main logs

Description

Get Opal main logs

Usage

oadmin.log(opal, all = TRUE)

Arguments

opal

Opal connection object.
all

Get all or only latest log messages.

Examples

## Not run: 
o <- opal.login('administrator','password', url='https://opal-demo.obiba.org')
oadmin.log(o)
opal.logout(o)

## End(Not run)
