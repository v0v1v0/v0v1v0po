sMatrix R Documentation

Similarity matrix.

Description

Construct a similarity matrix between 2 vectors.

Usage

sMatrix(a, b, filter = function(f1, f2) ifelse(f1 == f2, 1, -1))

Arguments

a

Vector a.
b

Vector b.
filter

A similarity function. Default: 1 if equal, -1 otherwise.

Value

A similarity matrix.
