chinook_customers R Documentation

Chinook customer data

Description

Extract of data on customers of a music sales company

Usage

chinook_customers

Format

A dataframe with 59 rows and 4 variables:

CustomerId

Customer ID number

FirstName

Customer First Name

LastName

Customer Last Name

SupportRepId

ID of Sales Rep assigned to customer

Source

Chinook Database on Github

Examples

chinook_customers
