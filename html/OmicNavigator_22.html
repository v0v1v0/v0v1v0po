cam.BasalvsML from Bioconductor workflow RNAseq123

Description

A subset of the object cam.BasalvsML from Bioconductor workflow RNAseq123.

Usage

cam.BasalvsML

Format

A data frame with 4 rows and 4 columns:

NGenes

Number of genes in each term

Direction

Direction of the enrichment

PValue

Nominal p-value

FDR

Multiple-testing adjusted p-value
