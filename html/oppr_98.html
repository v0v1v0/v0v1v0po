new_id R Documentation

Identifier

Description

Generate a new unique identifier.

Usage

new_id()

Details

Identifiers are made using the uuid::UUIDgenerate().

Value

Id object.

See Also

uuid::UUIDgenerate().

Examples

# create new id
i <- new_id()

# print id
print(i)

# convert to character
as.character(i)

# check if it is an Id object
is.Id(i)
