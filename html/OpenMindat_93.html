mindat_query

Description

Basic function for query dataset at a specified endpoit.

Usage

mindat_query (endpoint, query = list())

Arguments

endpoint

query endpoint, e.g.'minerals_ima'.
query

list for query conditions.

Value

df query resutls in data frame format.

Examples

## Not run:
