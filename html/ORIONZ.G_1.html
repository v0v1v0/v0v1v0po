ORIONZ.G-package R Documentation

EAP scoring in exploratory FA solutions with correlated residuals

Description

ORIONZ.G implements a procedure for obtaining Bayes EAP score estimates based on extended Exploratory Factor Analysis for both linear and nonlinear solutions with a residual structure.

Details

For more information about the methods used, please go to main page ORIONz.
